# SVWB デッキトラッカー

Shadowverse: Worlds Beyondの対戦中に、自分の山札の残り枚数やカードの種類をリアルタイムで手動管理・把握するためのデスクトップ向けWebアプリケーションです。

## 開発の背景・目的
試合中に山札の残りリソースを正確に把握し、プレイの選択肢を最適化することを目的として作成しました。規約違反のリスクがある自動読み込みツールを避け、手動更新型にすることで安全かつ実用的なツールとして要件を定義しました。

## 主な機能
- **カードの種類別エリア分け**: フォロワー（オレンジ）、スペル（水色）、アミュレット（黄色）で視覚的に分類
- **残り枚数に応じた視覚効果**: 残り1枚で黄色、0枚で赤色にカード名が変化
- **特殊カード対応処理**: 「試練の石板」発動時、山札の重複カード（2枚以上）をワンボタンで全て1枚にするロジックを搭載
- **永続化機能（プリセット保存）**: 最大5つまでのデッキ構成を保存・読込・削除可能。PC再起動後も状態を保持

## 開発プロセスと技術スタック
- **技術スタック**: Python / Streamlit / Pandas
- **開発手法**: 
  自身でシステムの「要件定義」「UI/UX設計」「データ構造の設計」を主導し、実装フェーズにおいては生成AI（LLM）を高度なコーディングアシスタント（ツール）として活用して短期間でプロトタイプを構築しました。
